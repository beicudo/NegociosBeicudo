git clone https://github.com/beicudo/NegociosBeicudo.git

git status
mostra o status do git, se não tem arquivos commitados ou modificados.

git add comandos_git.txt
adiciona o arquivo comandos_git.txt na área de stagging

git add .
adiciona todos os arquivos alterados ou criados para área de stagging
